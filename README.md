
# Project Title
Qr Code Generator


## Run Locally

Clone the project

```bash
  git clone https://github.com/OrhanEmirhanUlkatan/qr-code-generator.git
```

Go to the project directory

```bash
  cd qr-code-generator
```

Install dependencies

```bash
  npm install || yarn install
```

Start the server

```bash
  npm run dev || yarn dev
```
